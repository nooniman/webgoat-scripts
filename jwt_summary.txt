CHALLENGE 1: "Become Admin"
----------------------------

------------------------------------
Solution: Tamper reset token username

Steps:
1. Request password reset for yourself
2. Capture reset JWT token
3. Tamper: username: you ‚Üí username: admin
4. Submit with new password

-------------------------------

------------------------------------
Solution: Inject SQL in username field

Steps:
1. Get JWT token
2. Tamper: sub: user ‚Üí sub: admin' OR '1'='1
3. Submit

PowerShell: Option 4

--------------------------------
Solution: Brute force secret, sign as admin

Steps:
1. Get JWT token
2. Brute force secret
3. Sign new token: {"sub": "admin", "admin": true}
4. Submit

PowerShell: Option 8, then Option 9
HTML: Weak Secrets tab

===============================================
  TROUBLESHOOTING
===============================================

Problem: PowerShell won't run script
Solution: Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Bypass

Problem: Token not working
Solution: Try algorithm "none" or remove signature

Problem: "Invalid JWT format"
Solution: Ensure token has exactly 2 dots (header.payload.signature)

Problem: "Unauthorized"
Solution: Check header format: "Bearer TOKEN" not "JWT TOKEN"

Problem: Can't crack secret
Solution: Use external tools (jwt_tool, hashcat)
