âœ… Cookie is HttpOnly
   - JavaScript console CANNOT access it (document.cookie returns "")
   - Must edit manually in DevTools Application > Cookies tab
   - This is a security feature, not a bug

âœ… Algorithm "none" Attack
   - Change header algorithm from "HS512" to "none"
   - Remove signature (token ends with dot)
   - Server accepts token without signature verification

âœ… Correct Process
   1. F12 > Application > Cookies > Find access_token
   2. Copy current token (for timestamp reference)
   3. Double-click VALUE column
   4. Delete and paste attack token
   5. Press Enter
   6. Click "Delete" button or refresh page

âœ… Token Format
   - Must end with dot: ...payload.
   - Algorithm must be lowercase "none"
   - Payload must have admin:"true" (string, not boolean)
